# Project

Nijiral

## About

Nijira is an application for creating and managing projects collaboratively. Tasks can be assigned to individuals of the same workspace as members can be invited via an invite-link to a workspace.

### Technologies

- NextJs
- TypeScript
- TailwindCSS
- React Query
- React Hook Form
- React Table
- Zod
- Nuqs
